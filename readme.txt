== Description ==
###概要

ハローワークを利用して採用・求人活動を行っている企業にオススメ。

このプラグインでは、ハローワークインターネットサービスの求人情報から採用ページを簡単に作成できます。
一度作成しておけば、ハローワークの情報にあわせて自社の求人情報を自動的に取得・更新します。

つまり、今まで面倒だったハローワークに求人を出したり、求人募集を終了したりするたびに、自社サイトの採用ページを更新する手間が不要になります。

*細かな設定が不要な場合は、プラグインよりも[ウェブ版求人生成ジェネレータ](https://next.hellowork-jobs.info/op/company_url)が便利です。

###機能

- ハロ-ワークの求人情報を自動的に取得し、求人情報を生成
- ハローワークにて求人の更新や募集の終了を行った場合は、それに合わせて自社サイト内の求人情報を更新
